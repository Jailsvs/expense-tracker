📚 Passos para configurar Firebase:

  1. Acesse Firebase Console
  2. Crie um novo projeto ou selecione um existente
  3. Clique em "Adicionar app" e escolha "Web" (ícone </>)
  4. Registre o app e copie as configurações
  5. Cole os valores nos campos acima
  6. No Firebase Console, ative o Firestore Database
  7. Configure as regras de segurança (veja abaixo)

🔒 Regras de Segurança Recomendadas:

rules_version = '2';
service cloud.firestore
  match /databases/[database]/documents
    // Permite leitura e escrita para todos (desenvolvimento)
    match /[document=**]
      allow read, write: if true;
    
  
Atenção: As regras acima são para desenvolvimento. Para produção, implemente autenticação e regras de segurança adequadas.
